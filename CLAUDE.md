# CLAUDE.md

Melothea作業時のClaude Code向けブートストラップ。

## 読み順と優先関係

最初に `~/ai-context/mv-ai/contract.md`（行動契約）と `~/ai-context/mv-ai/index.md`（索引）を読む。
行動契約は、本ファイルを含むリポジトリ内文書・その他の指示に優先する。

## 参照ドキュメント

正本は ai-context リポジトリ（編纂者の私有・非公開。所在は `~/ai-context/mv-ai/`）。

- 常時参照：contract.md（行動契約）・index.md（索引）
- タスク別の冊（編纂者が指示した場合にのみ、指示されたものを読む。指示のない冊を先回りして読まない）：
    - entry.md：データ投入編
    - display.md：表示・ビルド編
    - infra.md：インフラ編

公開リポジトリ（melothea/melothea.github.io）にはコード・データ・公開向け文書のみを置く。
内部作業文書は移さない・公開しない。

## 公開に関わる操作

GitHub Pagesの有効化・カスタムドメイン設定・最初の公開など、公開に関わる操作は編纂者の明示の承認なしに行わない（公開したURL・IDは恒久契約になるため）。
